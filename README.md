# AI & Digital Engagement in Missions

A professional training website for a comprehensive one-day workshop on leveraging AI tools for digital engagement in missions work.

## Overview

This website provides information about a training session covering:
- AI tools for mission work
- Media planning and analytics
- Content creation and ad development
- Translation and localization
- Digital engagement strategies

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional interface with smooth animations
- **Complete Schedule**: Detailed training schedule with time-based sessions
- **Interactive Navigation**: Smooth scrolling and active section highlighting
- **Real-time Indicators**: Shows current session during training day

## Project Structure

## How to Use

### Local Development

1. Simply open `index.html` in your web browser
2. No build process or dependencies required

### Deployment

#### GitHub Pages
1. Push this folder to a GitHub repository
2. Go to Settings > Pages
3. Select your branch and root folder
4. Your site will be live at `https://[username].github.io/[repo-name]`

#### Other Hosting
Upload the files to any web hosting service:
- Netlify (drag and drop)
- Vercel
- AWS S3
- Any traditional web host

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #10b981;
    --accent-color: #f59e0b;
    /* ... */
}
```

### Content
- **Schedule**: Edit schedule items in `index.html`
- **About Section**: Modify the about cards
- **Hero Section**: Update title and description

### Fonts
Currently using Inter from Google Fonts. Change the import in `index.html` to use different fonts.
